CW puzzle formats:
* PUZ (*.puz) - binary (NY Times):  https://code.google.com/archive/p/puz/wikis/FileFormat.wiki
* IPUZ (*.ipuz) - json (open): http://www.ipuz.org/ 
* XPF (*.xpf) - xml (NY Times): https://www.xwordinfo.com/XPF/
* NYTJSON (*.nytjson, *.json) - json (NY Times): https://www.xwordinfo.com/JSON/
* YACDF (*.yacdf, *.yaml) - yaml (Financial Times): https://labs.ft.com/experiment/2018/03/26/crossword-data-format.html
